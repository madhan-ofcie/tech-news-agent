
# 🤖 Tech News Content Agent

An AI-powered autonomous agent that generates daily tech news scripts and YouTube thumbnails, delivered to your inbox every morning at 6 AM IST.

**Built for the AWS Builder Center Weekend Agent Challenge 2026**

---

## 🎯 What It Does

This agent runs **completely on its own** — no button clicks needed!

Every day at 6:00 AM IST, it automatically:
1. 📝 Generates a professional 2-3 minute tech news script using AI
2. 🎨 Creates a catchy thumbnail title (4-5 words, ALL CAPS)
3. 🖼️ Generates a YouTube-ready thumbnail image (1280x720)
4. ☁️ Stores everything in Amazon S3
5. 📧 Sends you an email with the script + thumbnail attached

**You wake up → Content is ready → Start recording!**
